# NotebookLM Terminal SOP 🧠

Use this folder to store raw outputs from your terminal brainstorming sessions. This keeps your "Ideas" separate from your "Scripts".

## 🚀 Commands to run from Terminal
Run these from the `Cook/` root:

### 1. Generate Content Ideas (Daily)
```bash
py -3 -m notebooklm ask --notebook YOUR_ID "Generate 3 reel topics for Personal Branding for a Sales Rep at Savasa who is also a Binus student" > "PDCT_JO_Consult/B Writing/3 LM Files/Topic_Ideas.md"
```

### 2. Scripting from Research
```bash
py -3 -m notebooklm generate YOUR_ID report > "PDCT_JO_Consult/B Writing/3 LM Files/Research_Report.md"
```

## 📋 The "Stitch" Workflow
1. **Brainstorm:** Get raw output in this folder.
2. **Draft:** Move the best ideas to `B Writing/2. Format/Reels/`.
3. **Stitch:** Reference the B-roll Aliases from `C Storyboarding/0. Asset/_Asset_Index.md` inside your script.

*Example Script line:*
> "Showing `[SAVASA_HANAMI_01]` while I explain why Japanese tech is the future of property."
