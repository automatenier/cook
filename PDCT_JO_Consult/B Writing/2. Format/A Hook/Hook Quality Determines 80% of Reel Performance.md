# Hook Quality Determines 80% of Reel Performance

## The Idea
The first 1–3 seconds of a reel determine whether someone stops scrolling. Everything else — edit quality, sound, CTA — is secondary. A great hook with average editing outperforms a great edit with a weak hook every time.

## Evidence
- Swipe file analysis: viral reels share pattern-interrupt hooks
- Tool: `tools/analyze_viral_reel.py` — hook patterns are the most common signal

## Hook types that work (fitness niche)
1. Contradiction: "Stop doing X if you want Y"
2. Curiosity gap: "Most coaches miss this one thing..."
3. Identity challenge: "If you're a serious coach, you already know..."
4. Specific number: "3 reels that got my client 1,200 followers in a week"
