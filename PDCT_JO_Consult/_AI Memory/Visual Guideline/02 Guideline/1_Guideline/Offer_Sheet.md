### 3 — Mesin Konten AI (AI Content Engine)

Ini bukan kursus tentang cara bikin konten. Ini sistem yang memproduksi konten untuk Anda — tiap bulan.

**Triple-layer AI pipeline:**
```
Claude (script) → ElevenLabs (voiceover) → Remotion (video draft) → CapCut (finishing ~15 mnt)
```

**Output bulanan:**

- **30 Reels/bulan (15 Authentic + 15 Value & CTA)**
  Script di-generate AI dari swipe structures terbukti viral yang sudah dianalisis. ElevenLabs clone suara Anda untuk segmen non-talking-head. Remotion render text/motion layer → Anda tinggal polish di CapCut ~15 menit per reel (bukan 45 menit dari nol).

- **30 Story Sequences/bulan**
  14 tipe stories yang berputar: testimoni, mendobrak nilai, cerita personal, kemenangan, CTA lembut, lifestyle. AI generate slide per slide, siap posting.

- **30 Threads Posts/bulan (1 per hari)**
  Di-generate dari winning outliers — format terbukti yang dianalisis AI. Mix: thread cerita, tips, hot take, studi kasus, behind-the-scenes. CTA arahkan ke lead magnet atau YouTube.

- **4 YouTube Scripts/bulan**
  Full script dari winning outlines — termasuk 3 opsi judul, thumbnail text, timestamps, riset keyword. Bukan outline, tapi script siap rekam.

- **TikTok Mirroring — 30 Reels + 30 Stories**
  Semua konten IG di-package ulang untuk TikTok: caption lebih pendek, hashtag TikTok-native, CTA disesuaikan. Export CapCut tanpa watermark → upload TikTok.

- **150+ Content Pieces/bulan dari Repurposing Engine**
  1 reel → dikerjakan AI menjadi: caption IG, caption TikTok, Threads post, story sequence, outline YouTube, newsletter snippet. Multiply output tanpa multiply waktu.

- **AI Swipe File — Sistem Memori Per Klien**
  Sebelum generate konten, AI membaca swipe file Anda — berisi brand voice, ICP, hooks terbaik, offer summary, dan pelajaran dari konten sebelumnya. Semakin lama program, semakin pintar sistem menghasilkan konten yang sesuai dengan Anda.

- **ElevenLabs AI Voice Clone (500 kata/batch)**
  Kita clone suara Anda dari rekaman 5-10 menit. Dipakai untuk voiceover di segmen non-talking-head, narasi stories, dan ad voiceover. Terdengar seperti Anda — tanpa harus rekam ulang.

- **Filming Onsite JABODETABEK (2 Jam)**
  Kita datang ke lokasi Anda. Shot list dibuat AI berdasarkan content calendar: talking head setups (5-8 angle berbeda), b-roll, VSL recording, testimonial clips, asset photos. Output: raw footage untuk ~3 bulan konten.

- **GDrive Content File Management System (10 GB)**
  Bukan folder kosong — sistem file yang sudah terstruktur dari Hari 0:
  ```
  GDrive/[Nama Kamu]/
  ├── A-Roll/          ← Talking head clips, terindeks per angle
  ├── B-Roll/          ← Lifestyle, workspace, activity clips
  ├── Assets/          ← Logo, grafik, AI-generated photos
  ├── Audio/           ← ElevenLabs voice output per batch
  ├── Edited/          ← Final reels siap posting (IG + TikTok)
  └── Posted/          ← Arsip konten yang sudah live
  ```
  Excel Footage Tracker terintegrasi: setiap clip tercatat (nama file, tipe, status, dipakai di reel mana). Tim tahu persis apa yang ada dan apa yang perlu direkam ulang.

- **Remotion Video Draft System**
  Text + motion layer di-render otomatis dari script — langsung bisa diimport ke CapCut. Finishing ~15 menit vs 45 menit dari nol. Hemat ~60% waktu editing per reel.
