Pakai Remotion

Panduan lengkap untuk online coach yang mau scale konten tanpa edit manual — satu template, ratusan video.

JO Consult × Mathew Jordan · 2026
Kebanyakan coach stuck karena harus edit satu-satu. Remotion membalik logika ini: lo build template sekali, sistem yang generate ratusan video dari data yang lo kasih.

Apa itu Remotion?

Remotion adalah tool yang bikin lo bisa render video pakai kode — bukan edit manual di CapCut atau Premiere. Lo nulis template sekali, lalu sistem yang generate video dari script yang lo kasih.

Untuk coach, artinya:

  • Nggak perlu duduk edit tiap reel satu per satu
  • Tim lo bisa produce 30 reel sebulan dari 1 template
  • Perubahan visual? Ganti di satu file, semua video auto-update

Yang Lo Butuhkan

  • Node.js terinstall (gratis — nodejs.org)
  • VS Code atau text editor apapun
  • Script reel yang sudah jadi (teks hook + isi + CTA)
  • Waktu setup awal: ~2 jam

Step-by-Step Setup

1

Install Remotion

Buka terminal / command prompt, ketik:

npx create-video@latest

Pilih template "Blank" untuk mulai dari nol, atau minta template yang sudah jadi dari tim lo.

2

Pahami Struktur Project

remotion-project/ ├── src/ │ ├── Root.tsx ← daftar semua video template │ ├── Reel.tsx ← template visual reel lo │ └── index.ts ├── public/ ← background video / gambar └── package.json

Yang paling penting: Reel.tsx — template visual yang dipakai untuk semua reel.

3

Buat Template Reel Sederhana

Contoh template text-over-video (paling umum untuk coach content):

// src/Reel.tsx import { AbsoluteFill, useCurrentFrame, interpolate } from 'remotion'; export const Reel = ({ hook, body, cta }) => { const frame = useCurrentFrame(); const opacity = interpolate(frame, [0, 20], [0, 1]); return ( <AbsoluteFill style={{ backgroundColor: '#000', padding: 60 }}> <p style={{ color: 'white', fontSize: 52, fontWeight: 'bold', opacity, lineHeight: 1.3 }}> {hook} </p> </AbsoluteFill> ); };
4

Masukkan Script Lo

<Composition id="Reel" component={Reel} durationInFrames={450} // 15 detik di 30fps fps={30} width={1080} height={1920} defaultProps={{ hook: "3 kesalahan coach yang bikin klien kabur", body: "Lo mungkin lakuin ini tanpa sadar...", cta: "Comment 'SCRIPT' untuk template DM gratis" }} />
5

Preview di Browser

npm run preview

Buka localhost:3000 — lo bisa lihat video real-time sambil edit teks.

6

Render Jadi Video

npx remotion render Reel out/reel-01.mp4

Video MP4 siap upload ke IG / TikTok.

Workflow Efisien: Batch Render dari Data

Ini yang bikin Remotion beda dari edit manual. Lo bisa render 30 video sekaligus dari spreadsheet.

1. Buat file scripts.json:

[ { "hook": "3 kesalahan coach baru", "body": "...", "cta": "..." }, { "hook": "Kenapa klien lo ghosting", "body": "...", "cta": "..." }, { "hook": "Formula DM yang closing", "body": "...", "cta": "..." } ]

2. Loop render dengan script Node.js:

node render-batch.js

Hasilnya: 30 file MP4 siap upload, dikerjain sistem — bukan lo.

Batasan yang Perlu Lo Tau

Heads up: Remotion butuh sedikit comfort dengan kode. Kalau belum pernah coding sama sekali, ada learning curve ~1 minggu. Render juga butuh laptop yang cukup kuat, atau pakai cloud rendering. Template awal butuh effort setup — tapi setelah jadi, sistem yang kerja.

Mau sistem ini jalan fully otomatis?

Dari script text ke video siap upload — tanpa render manual, tanpa editor duduk seharian. Itu yang kami build untuk klien Bedah Digital.
