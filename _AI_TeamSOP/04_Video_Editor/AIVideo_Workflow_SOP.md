# SOP: AI Video UI Animation (High-End)
Status: Stable
Category: Video Production

## Purpose
To create professional, snappy UI animations using AI models (Nano Banana Pro, Veo 3.1) and CapCut, reducing manual design and animation time.

## Models Used
1. **Asset Design:** Nano Banana Pro (for UI layers and product swaps).
2. **Animation:** Veo 3.1 (for physics-based UI motion).
3. **Refining:** CapCut (for speed ramping and final polish).
