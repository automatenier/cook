# Carousel Storyboard — SAVASA Deltamas
> Format: 1 sentence per slide | BG = background direction | Face = show face or not

---

## Personal Brand Visual System

| Element | Direction |
|---------|-----------|
| **Colors** | Dark navy `#0D1B2A` base · White text · Cyan accent `#00D4FF` for AI/data · Warm gold `#C9A84C` for property value |
| **Face style** | Casual, no tie — hoodie or clean tee. Not agent-mode. |
| **Face slides** | Always slide 1 (hook) + last slide (CTA). Optional 1 mid-carousel. |
| **Data slides** | NO face — big bold stat on dark bg |
| **Property slides** | Aerial or unit photo + dark overlay + white text |
| **AI slides** | Screenshot of actual AI chat + slight blur/overlay |
| **Fonts** | Bold heavy weight headline · Regular subtext |
