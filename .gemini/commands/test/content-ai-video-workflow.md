# Workflow: AI Video UI Animation

When user asks to `/ai-video-workflow`:

1. **Asset Generation (Nano Banana Pro)**: Use a Reference Image. Prompt Nano Banana Pro to break the design into layers. Goal: high-fidelity text and crisp UI elements.
2. **Animation (Veo 3.1)**: Prompt: "Create a smooth UI style animation for this design." Goal: AI handles physics and motion.
3. **Post-Production (CapCut)**: Apply Speed Ramping — fast at start, slow at end (ease-out effect).

## Execution Notes
- Nano Banana Pro and Veo 3.1 are accessed via InVideo or similar hosted platforms.
- If automated tools become available in `AI_Tools/`, update this workflow.
