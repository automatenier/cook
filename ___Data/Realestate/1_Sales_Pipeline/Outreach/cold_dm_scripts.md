# RE — Cold DM Scripts
> Language: Bahasa Indonesia | Platform: WhatsApp + Instagram DM
> Rule: Never send price before booking a viewing. Max 3 follow-ups per lead.

---

## INSTAGRAM DM — Cold Outreach

### Script IG-A (Curiosity Hook)
```
Hei [Nama]! Lagi cari properti di [area] ya?

Kebetulan ada unit yang baru masuk listing minggu ini — tipe yang biasanya sold dalam 2 minggu di area itu.

Boleh saya kirim detailnya?
```

### Script IG-B (Value-Led — Below Market)
```
Hei [Nama], perkenalkan — saya Jordan, konsultan properti.

Ada 1 unit di [area] yang harganya masih di bawah NJOP — owner lagi butuh dana cepat. Tipe kayak gini jarang banget.

Mau saya share speknya?
```

### Script IG-C (Social Proof)
```
Hei [Nama]! Klien saya kemarin baru closing unit di [area] — dapat harga bagus karena masuk duluan sebelum dipasarkan luas.

Kebetulan ada lagi 1 unit serupa yang belum dipublikasi. Tertarik lihat?
```

### Script IG-D (Referral)
```
Hei [Nama]! Dapat info dari [Nama Referral] kalau kamu lagi cari properti.

Saya bantu beberapa orang di [area] cari unit yang sesuai. Lagi cari tipe apa — untuk invest atau ditempati?
```

## WHATSAPP — Cold Outreach

### Script WA-A (Direct)
```
Halo [Nama]! Selamat [pagi/siang/sore].

Perkenalkan, saya Jordan — konsultan properti spesialis [area].

Saya lihat profil kamu dan mungkin ada unit yang cocok. Ada yang lagi dicari sekarang? Untuk invest atau tempat tinggal?
```

### Script WA-B (Content Lead — dari IG/TikTok)
```
Halo [Nama]! Tadi lihat kamu engage sama konten properti saya.

Kalau lagi cari unit di [area], ada beberapa pilihan menarik sekarang — sebelum harga naik lagi Q2 ini.

Boleh saya tanya dulu: budget range-nya sekitar berapa? Supaya saya bisa filter yang paling relevan.
```

### Script WA-C (Warm Network — Kenal Tapi Belum Pernah Tanya)
```
Halo [Nama]! Lama nggak ngobrol.

Saya lagi fokus di properti [area] sekarang. Kalau kamu atau siapapun yang kamu kenal lagi cari atau mau jual properti, happy banget kalau bisa bantu.

Gimana kabar? Masih di [lokasi terakhir]?
```

## QUALIFICATION QUESTIONS (setelah respons pertama)

Setelah lead reply, tanya dalam urutan ini:

1. "Untuk tempat tinggal sendiri atau investasi?"
2. "Kira-kira budget range-nya berapa? (agar saya bisa carikan yang paling sesuai)"
3. "Lokasi preferensinya di area mana? Ada pertimbangan khusus — dekat kantor, sekolah, dll?"
4. "Target kapan mau punya unit-nya? Tahun ini atau masih flexible?"
5. "Udah punya gambaran unit-nya — landed, apartemen, atau open?"

**Lead scoring:**
- HOT: budget jelas + timeline 1-3 bulan + lokasi spesifik
- WARM: budget rough + timeline 3-6 bulan + area flexible
- COLD: "masih lihat-lihat" + no budget + no timeline

## RULES

- Jangan kirim harga di DM pertama — selalu booking viewing dulu
- Kalau nggak reply dalam 3 hari → pindah ke follow-up scripts
- Max 3 follow-up per lead sebelum mark sebagai Cold
- Selalu catat sumber lead di tracker: `py -3 24_Tools/log_lead.py --name "[Name]" --source "IG Organic" --vertical re --status new`
