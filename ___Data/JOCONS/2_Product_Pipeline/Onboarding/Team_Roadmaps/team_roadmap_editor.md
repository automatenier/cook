# Team Roadmap — Editor

> SOP untuk Video Editor yang handle CapCut production, dari Remotion draft sampai final export.

---

## Role Summary

Editor menerima Remotion video drafts + scripts + voiceovers dari pipeline AI, lalu polish di CapCut menjadi final reels, stories, dan YouTube content. Output harus siap posting di IG + TikTok.

---

## 14-Day Sprint (Per Client Baru)

### Day 4-5 — VSL & Testimonial Editing
- [ ] Terima raw VSL footage dari filming session
- [ ] Edit VSL di CapCut:
  - Cut dead air, add transitions
  - Add lower thirds (dari `graphics/lower_thirds/`)
  - Add background music (subtle, professional)
  - Add chapter titles matching VSL script sections
  - Export 1080p → upload ke Wistia
- [ ] Edit testimonial clips (3-5 clips, 30-90 detik each):
  - Clean cuts, add client name lower third
  - Export → upload ke Wistia

### Day 7-10 — First Content Batch Production
- [ ] Terima dari VA:
  - 30 Remotion MP4 drafts (text/motion layer)
  - 30 reel scripts (15 Authentic + 15 Value-CTA)
  - ElevenLabs voiceover audio files (untuk non-talking-head reels)
  - Client's talking head clips (dari filming session)
  - Graphics guide (`graphics/color_palette.md`)

### CapCut Workflow Per Reel (~15 min each):
1. Import Remotion MP4 (text/motion layer)
2. Add b-roll footage ATAU talking head video
3. Jika non-talking-head: sync ElevenLabs voiceover audio
4. Add transitions + music dari winning CapCut template
5. Apply personal graphics:
   - Lower thirds (name/title overlays)
   - CTA graphics (keyword graphics)
   - Brand colors + fonts
6. Export for IG (9:16, high quality)
7. Export for TikTok (same project, no IG watermark)

### Day 8-10 — Story & Additional Content
- [ ] Edit 30 story slides di CapCut (dari story scripts)
- [ ] Apply branded story frame templates (`graphics/story_templates/`)
- [ ] Export stories for both IG + TikTok

---

## Content Feedback Loop (Creative Call)

> Setiap Creative Call, Creative Director review sistem konten dan beri arahan.

### Sebelum Creative Call:
- [ ] Siapkan 5 reels terbaru yang sudah di-edit (draft atau final)
- [ ] Catat pertanyaan / blocker yang kamu hadapi
- [ ] Pull metrics dari content yang sudah posting (views, engagement)

### Selama Creative Call:
- Creative Director review "Content Automation System"
- Data-driven review: analisis "Followers per 1,000 views" dan metrics lain
- Editor Management: review workflow antara client dan editor
- Dapat arahan untuk adjustment (pacing, music style, graphic tweaks)

### Setelah Creative Call:
- [ ] Implement feedback ke batch berikutnya
- [ ] Update CapCut template jika ada perubahan style

## Quality Checklist (Per Reel)

Before exporting, verify:
- [ ] Aspect ratio: 9:16 (1080x1920)
- [ ] Text readable on mobile (minimum 48pt for hooks)
- [ ] Audio levels balanced (voiceover > music, no clipping)
- [ ] Hook appears in first 0.5 seconds (no dead intro)
- [ ] CTA keyword graphic appears in last 3 seconds
- [ ] Lower third with client name/title appears
- [ ] Brand colors match `color_palette.md`
- [ ] No IG watermark on TikTok export
- [ ] Total duration: 15-60 seconds (sweet spot: 20-30s)
- [ ] File naming: `{client}_{type}_{week}_{number}.mp4`
  - Example: `coach_aldi_authentic_w1_03.mp4`
