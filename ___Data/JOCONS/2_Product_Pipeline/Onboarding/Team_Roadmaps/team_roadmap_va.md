# Team Roadmap — VA (Virtual Assistant)

> SOP harian untuk VA yang mengelola operasional, scheduling, dan koordinasi antar tim.

---

## Role Summary

VA adalah operational backbone. Tugasnya memastikan semua proses berjalan on-time, semua file terorganisir, dan semua orang punya apa yang mereka butuhkan.

---

## 14-Day Sprint (Per Client Baru)

### Day 0 — Client Signs
- [ ] Create client folder structure di GDrive (10GB shared folder)
- [ ] Kirim welcome email (dari template yang sudah Claude generate)
- [ ] Kirim link Onboarding Form ke client
- [ ] Add client ke WhatsApp group + Telegram community
- [ ] Share 20+ video module access
- [ ] Setup Telegram progress channel via n8n
- [ ] Update CRM → stage: "Closed"

### Day 1-3 — Onboarding Support
- [ ] Follow up client untuk lengkapi Onboarding Form (jika belum)
- [ ] Follow up client untuk upload materials ke GDrive (testimonials, photos, voice recording, Client Info docs)
- [ ] Organize incoming materials ke folder yang benar:
  - `testimonials/`, `achievements/`, `brand_voice_samples/`, `voice_clone/training_samples/`
- [ ] Share asset handover ke client (training PDF, nutrition guide, DFY assets, roadmaps)
- [ ] Confirm onboarding data complete → notify Jordan untuk run offer agent

### Day 3-5 — Funnel & Filming Coordination
- [ ] Schedule onsite filming session (coordinate waktu + lokasi JABODETABEK)
- [ ] Kirim filming prep brief ke client (outfit, location prep, shot list dari Claude)
- [ ] After filming: organize raw footage ke GDrive (`raw_footage/` folder)
- [ ] Upload edited VSL ke Wistia → catat media ID
- [ ] Upload testimonial clips ke Wistia → catat media IDs
- [ ] Deploy Netlify funnel → test semua links (VSL playback, WhatsApp CTA, calendar link, mobile)

### Day 5-7 — Content Calendar
- [ ] Collect client's last 20 posts (links + screenshots + metrics)
- [ ] Share approved content calendar ke client via GDrive / Notion

### Day 7-10 — Content Production
- [ ] Organize generated scripts ke folder yang benar (`scripts/`, `remotion_props/`)
- [ ] Coordinate dengan Editor: share Remotion MP4s + voiceovers + scripts
- [ ] Track content production progress (drafted / editing / approved / posted)

### Day 8-10 — CTA & Ads Setup
- [ ] Setup IG keyword CTA di Meta Business Suite (keyword pairs dari Claude)
- [ ] Setup TikTok keyword CTA (bio link, pinned comments)
- [ ] Request Meta Business Suite access dari client (Admin role)

### Day 10-12 — Lead Activation Support
- [ ] Help client export old lead list
- [ ] Organize leads into Hot / Warm / Cold segments (spreadsheet)
- [ ] Track reactivation responses di CRM

### Day 13-14 — Consultation Prep
- [ ] Prepare all deliverables for review (link semua di satu doc)
- [ ] Setup Chrome "Sales HQ" bookmarks for client
- [ ] Create Google Sheets "Command Center" dari template Claude
- [ ] Schedule 30-min walkthrough call after consultation

## Daily Routine (Ongoing Post Day 14)

| Time | Task |
|---|---|
| **08:00** | Check Telegram channels — flag overdue items |
| **08:30** | Update CRM with yesterday's activities (DMs, leads, content posted) |
| **09:00** | Coordinate with Editor — confirm today's batch is on track |
| **09:30** | Coordinate with Setter — review yesterday's EOD report |
| **10:00** | Follow up on client responses / pending approvals |
| **12:00** | Check content posting schedule — confirm scheduled posts went live |
| **14:00** | Organize incoming assets / feedback from client |
| **16:00** | Prep tomorrow's tasks — brief Editor + Setter if needed |
| **17:00** | Send EOD summary to Jordan via Telegram |

---

## Weekly Tasks

| Day | Task |
|---|---|
| **Monday** | Distribute weekly content batch to Editor (scripts + voiceovers + Remotion files) |
| **Wednesday** | Mid-week check: content production status, setter KPIs |
| **Friday** | Compile weekly report: content posted, leads generated, calls booked |
| **Sunday** | Review next week's content calendar — flag any gaps |

---

## KPIs

| Metric | Target |
|---|---|
| Onboarding completion | Within 48 hours of payment |
| Client materials organized | Within 24 hours of receipt |
| Content batch delivered to Editor | Every Monday by 10:00 |
| CRM updated | Daily by 17:00 |
| Response to client messages | Within 4 hours during business hours |
