## 1. Inbound Script

**Trigger:** Lead DM kamu duluan (dari keyword CTA, story reply, atau langsung tanya).

### Step 1 — Acknowledge + Personalize
```
"Hey [nama]! Thanks udah reach out 🙏
Gue liat kamu [detail spesifik dari profil mereka — e.g., coach di Surabaya / udah punya 3K followers].
Apa yang bikin kamu tertarik [keyword / topic]?"
```

### Step 2 — Discovery (3-5 pertanyaan)
```
"Nice! Jadi sekarang [bisnis/coaching] kamu lagi di stage mana?"
"Udah berapa lama run online coaching?"
"Yang paling struggle apa sekarang? Leads, content, atau closing?"
"Kamu udah pernah coba [solution] sebelumnya?"
```

### Step 3 — Validate Pain
```
"Gue paham banget — itu literally yang paling sering gue dengar dari coach di stage kamu."
"[Nama client] juga dulu di posisi yang sama, sekarang dia udah [result]."
```

### Step 4 — Bridge to Call
```
"Honestly ini lebih enak dibahas via call — gue bisa liat situasi kamu lebih detail
dan kasih tau exactly gimana kita bisa bantu.
Mau gue kirimin link buat book 15 menit?"
```

### Step 5 — Booking
```
"Perfect — ini link-nya: [Calendar Link]
Pilih waktu yang paling enak buat kamu.
Setelah booking, kamu bakal terima reminder di WA juga."
```

**Jika mereka tanya harga duluan:**
```
"Good question — tergantung situasi kamu sih.
Ada beberapa opsi dan gue mau kasih yang paling pas.
Itu kenapa mending ngobrol dulu 15 menit — biar gak salah recommend.
Fair enough?"
```

## 2. Outbound Script

**Trigger:** Kamu initiate kontak ke ICP yang belum DM.

### Phase A — Warm Up (2-3 hari SEBELUM DM)
1. Follow akun mereka
2. Like 3-5 post terbaru (genuine, bukan spam)
3. Comment meaningful di 2 post
4. Watch + react stories mereka
5. Reply ke story secara conversational

### Phase B — Open DM
```
"Hey [nama]! Gue liat konten kamu tentang [topic spesifik] — solid banget.
Boleh nanya, itu [question about their content/approach]?"
```

**Alternative openers:**
```
"[Nama], transformation client kamu yang [detail] itu gila sih. Berapa lama prosesnya?"
```
```
"Gue notice kamu coaching [niche] — kamu fokus online atau offline?"
```

### Phase C — Build Rapport (3-5 exchanges)
- Talk about THEM
- Match energy mereka
- Jangan mention apa yang kamu jual
- Tanya tentang bisnis, journey, goals mereka

### Phase D — Surface Pain
```
"Btw, sekarang lagi fokus grow ke mana nih bisnisnya?"
"Yang paling challenging apa di [area mereka mention]?"
```

### Phase E — Transition (hanya kalau mereka share pain)
```
"Interesting — gue actually bantu coach di situasi kayak gitu.
Mau gue share gimana [client name] solve problem yang sama?"
```

→ Kalau "ya" → kirim case study / voice note → bridge to call
→ Kalau lukewarm → nurture (kirim content, stay in touch)

## 3. Post-Booking Script

**Trigger:** Lead baru booking call. Kirim SEGERA.

### Immediately After Booking
```
"Ayy [nama]! Udah masuk booking kamu 🎯
Excited banget buat ngobrol [hari/tanggal].

Quick heads up — biar call-nya maksimal:
1. Siapin pertanyaan tentang bisnis kamu sekarang
2. Pikirin target realistis 3-6 bulan ke depan
3. Kalau ada, share link IG coaching kamu biar gue bisa research dulu

See you soon! 🤝"
```

### 24 Hours Before Call
```
"Hey [nama]! Reminder — besok kita call jam [waktu].
Gue udah cek profil kamu dan ada beberapa insight yang mau gue share.
See you there 💪"
```

### 1 Hour Before Call
```
"[Nama] — 1 jam lagi kita connect!
Make sure link Zoom/Google Meet kamu ready ya.
Kalau ada perubahan, kabarin gue."
```

### Voice Note (Night Before) — PERSONAL
> "Hey [nama], ini [setter name]. Gue udah research akun kamu dan honestly ada beberapa hal keren yang bisa kita bahas besok. Gue excited buat call-nya — see you besok!"

## 4. Freebie Script

**Trigger:** Lead comment keyword CTA atau request free resource (PDF, checklist, video).

### Step 1 — Deliver Immediately
```
"Hey [nama]! Ini dia [resource name] yang kamu request 🎁
[Link / File]

Semoga bermanfaat! Gue curious — kamu lagi di stage mana sekarang coaching-nya?"
```

### Step 2 — Follow Up (24 hours later, kalau mereka respond)
```
"Gimana, udah sempet cek [resource]-nya?
Ada bagian yang paling relate sama situasi kamu?"
```

### Step 3 — Discovery
```
"Nice! Btw kamu mention [their response] —
itu actually area yang sering banget kita bantu coach solve.
[Nama client] dulu juga struggle di situ, sekarang [result].
Mau gue share lebih detail gimana prosesnya?"
```

### Step 4 — Bridge to Call
```
"Honestly ini lebih enak dibahas 15 menit via call.
Gue bisa kasih breakdown spesifik buat situasi kamu.
Mau book? [Calendar Link]"
```

**Jika TIDAK respond setelah freebie:**
```
(3 hari kemudian)
"Hey [nama]! Gimana [resource name]-nya — udah helpful?
Kalau ada pertanyaan, feel free tanya di sini 🙏"
```

## 5. Inbound Ads Script

**Trigger:** Lead masuk dari Meta Ads (biasanya sudah agak warmed up dari ad creative).

### Step 1 — Fast Response (< 15 menit)
```
"Hey [nama]! Thanks udah [action dari ad — daftar / klik / isi form].
Gue liat kamu tertarik sama [topic dari ad].
Boleh nanya — kamu udah jalanin online coaching atau masih planning?"
```

### Step 2 — Quick Qualify (2-3 pertanyaan max)
```
"Nice! Dan sekarang yang paling jadi challenge apa?"
"Kamu lagi actively cari solusi atau masih riset dulu?"
```

### Step 3 — Fast Track to Call
```
"Got it — honestly dari apa yang kamu ceritain, ini persis yang kita bantu.
Mau gue set up 15 menit call?
Biar kamu bisa denger langsung gimana prosesnya dan decide apakah cocok.
[Calendar Link]"
```

**Key difference dari Inbound biasa:**
- Ads leads expect FASTER response
- Mereka udah exposed ke messaging — skip long rapport building
- Go straight to qualify → book
- Tapi tetap CONVERSATIONAL, bukan robotic

## 6. Show-Up Structure

**Trigger:** Prospect sudah booking — ini sequence untuk maximize show rate.

### Timeline

| Waktu | Action | Channel |
|-------|--------|---------|
| **Booking confirmed** | Auto-confirmation email + WA | n8n automation |
| **24 jam sebelum** | Personal DM reminder | Instagram DM |
| **24 jam sebelum** | Physical phone call | Phone |
| **Night before** | Personal voice note | WA / IG Voice |
| **1 jam sebelum** | Quick text reminder | WA |
| **15 menit sebelum** | "Ready when you are" | WA |

### 24-Hour Phone Call Script
```
"Hey [nama], ini [setter name] dari JO Consult.
Gue mau confirm aja — besok kita call jam [waktu] ya?
Udah siap? Ada yang mau kamu tanyain duluan?"
```

### If They Sound Hesitant
```
"No worries — ini cuma ngobrol santai, bukan sales pitch.
Kita mau liat apakah kita bisa bantu atau nggak.
Kalau ternyata gak cocok, no hard feelings.
Fair enough?"
```

### No-Show Protocol
- Wait 5 minutes → WA: "Hey [nama], kita harusnya call sekarang — masih jadi?"
- Wait 15 minutes → DM: "No worries kalau ada something! Mau reschedule?"
- Wait 24 hours → Follow-Up Script (below)

## 7. Follow-Up Structure

**Trigger:** No response, no show, atau conversation yang jalan tapi belum booking.

### Type A — No Show Follow-Up
```
(Same day, 2-3 jam setelah missed call)
"Hey [nama]! No worries tentang tadi — things happen 🙏
Mau reschedule? Gue masih available [2-3 slot].
[Calendar Link]"
```

```
(2 hari kemudian, kalau no response)
"[Nama] — just checking in. Masih interested ngobrol?
Kalau sekarang belum timing yang tepat, totally fine.
Gue kirim [valuable resource] aja dulu yang mungkin helpful."
```

### Type B — Ghosted After Good Convo
```
(3-5 hari setelah last message)
"Hey [nama]! Gue inget kamu cerita tentang [pain point mereka].
Baru aja ada case study yang PERSIS situasi itu — [brief teaser].
Mau gue share?"
```

```
(7 hari, final attempt)
"[Nama] — one last thing.
Gue tau kamu sibuk, tapi kalau [pain] masih jadi issue,
gue ada 15 menit minggu ini buat ngobrol.
No pressure — just thought I'd offer.
[Calendar Link]"
```

### Type C — Said "Nanti Aja" / "Belum Siap"
```
(14 hari kemudian)
"Hey [nama]! Pas terakhir ngobrol, kamu bilang [quote mereka].
Curious — ada progress di [area mereka]?
Kalau belum, gue punya [resource] yang mungkin bantu."
```

```
(30 hari — long game nurture)
Share content yang relevant (YouTube, reel, article) tanpa CTA.
Biarkan mereka come back naturally.
```

### Follow-Up Rules
| Rule | Detail |
|------|--------|
| Max follow-ups | 3x (setelah itu → nurture mode) |
| Tone | Helpful, NEVER desperate |
| Always add value | Setiap follow-up harus bawa sesuatu (insight, resource, case study) |
| Never say | "Just checking in", "Following up", "Bumping this" |
| Long game | Kirim YT/content setiap 2 minggu — mereka akan come back |

## 8. Upsell Script

**Trigger:** After successful close, or for existing clients eligible for upgrade.

### Type A — Post-Close Referral Ask
```
(7 hari setelah onboarding, setelah mereka excited)
"Hey [nama]! Gimana so far experience-nya?
Btw — kamu kenal coach lain yang struggle hal yang sama kayak kamu dulu?
Kalau mau introduce, gue bisa kasih mereka free strategy session.
Dan buat kamu ada [referral reward] juga 🎁"
```

### Type B — Tier Upgrade
```
(Setelah client deliver results di paket awal)
"[Nama] — congrats buat progress kamu! 🔥
Gue liat kamu udah [achievement].
Kita punya [next tier/add-on] yang bisa bantu kamu scale ke [next level].
Mau gue explain gimana itu work?"
```

### Type C — Add-On Service
```
"Btw [nama] — banyak client kita yang di stage kamu juga ambil [add-on: ads management / extra content / team hiring support].
Hasilnya [specific result].
Mau gue kasih detail?"
```

## DM Formatting Rules

| Rule | Detail |
|------|--------|
| **Message length** | Max 3-4 baris per bubble. Jangan wall of text. |
| **Emoji** | Sparingly. Max 1-2 per message. |
| **Voice notes** | USE THEM. More personal than text. 30-60 detik max. |
| **Response time** | < 15 menit saat jam kerja. < 1 jam di luar jam kerja. |
| **Bahasa** | Match prospect. Formal → formal. Casual → casual. |
| **NEVER** | Copy-paste tanpa personalize. Prospect TAHU template. |
| **ALWAYS** | Mention something SPECIFIC from their profile/content. |

## Objection Quick Reference

| Objection | Handle |
|-----------|--------|
| "Berapa harganya?" (too early) | "Tergantung situasi — mending ngobrol dulu 15 min biar gak salah recommend." |
| "Gak punya waktu" | "1-2 jam syuting = stok 1 bulan. Tim kita handle sisanya." |
| "Udah pernah coba, gak work" | "Makes sense — yang bedain itu [unique mechanism]. Mau gue tunjukin?" |
| "Nanti aja deh" | "No pressure. Gue kirim [resource] dulu — kapanpun ready, hit me up." |
| "Kemahalan" | "Kalau subscribe semua tools sendiri = jutaan/bulan. Kita pangkas itu." |
| "Tanya partner dulu" | "Totally — mau gue siapin summary yang bisa kamu share ke mereka?" |

## Disqualification Script

Kalau lead JELAS bukan ICP, exit dengan grace:
```
"Hey [nama], thanks udah ngobrol!
Honestly dari apa yang kamu ceritain, gue rasa timing-nya belum pas buat program ini.
Tapi gue ada [free resource] yang mungkin bantu di stage kamu sekarang.
[Link]
Kalau nanti ada yang berubah, feel free reach out lagi! 🙏"
```
