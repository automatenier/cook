# Content Gap Analysis Template

> Identifies missing content pillars, format gaps, and CTA weaknesses. Run AFTER the content audit checklist.

---

## Input Required
- Completed content audit checklist (20 posts scored)
- Client swipe file
- Client's ICP description

---

## 1. Pillar Distribution Analysis

Count how many of the 20 posts fall into each pillar:

| Pillar | Count (/20) | % | Ideal % | Gap |
|---|---|---|---|---|
| Authority (proof, expertise) | | | 20% | |
| Connection (personal, BTS) | | | 20% | |
| Education (tips, how-to) | | | 25% | |
| Social Proof (testimonials) | | | 15% | |
| CTA/Conversion (offers, lead magnets) | | | 20% | |

### Diagnosis:
- **Missing pillar:** Any pillar with 0 posts = critical gap
- **Over-indexed:** Any pillar > 40% = over-reliance, audience fatigue risk
- **Under-indexed:** Any pillar < 10% = missed opportunity

### Common Patterns:
| Pattern | Problem | Fix |
|---|---|---|
| All Education, no CTA | "Free coach" — gives value but never converts | Add keyword CTA to every 3rd post |
| All Authority, no Connection | Looks impressive but not relatable | Add 2 personal/vulnerability posts per week |
| All Connection, no Social Proof | Likable but no credibility | Add 1 testimonial post per week |
| No Education at all | Not building trust with cold audience | Add 2 educational posts per week |

## 3. CTA Strategy Audit

| Question | Answer |
|---|---|
| Does the client use keyword CTAs? | Yes / No |
| How many posts have a clear CTA? | ___/20 |
| Where do CTAs lead? | Link in bio / DM / Lead magnet / Nothing |
| Is Meta Biz Suite keyword automation set up? | Yes / No |
| Do they have a lead magnet? | Yes / No |
| Do they have a VSL funnel? | Yes / No |

### CTA Gap Score:
- 0-2 CTAs out of 20 posts = **Critical** — no conversion mechanism
- 3-6 CTAs = **Weak** — sporadic, no system
- 7-12 CTAs = **Moderate** — has intent but inconsistent
- 13-20 CTAs = **Strong** — systematic approach
